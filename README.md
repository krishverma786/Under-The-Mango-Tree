# Under The Mango Tree - Premium Restaurant Web Application

A fully custom-coded, full-stack responsive web application designed for a premium restaurant brand. This project replaces generic templates and third-party booking intermediaries with a seamless, commission-free digital storefront and an intuitive, secure data management ecosystem.

## ✨ Key Features

### 🍽️ Client-Facing Portal
* **Dynamic Landing Page:** An elegant, modern hero landing structure tailored to showcase brand aesthetics and drive conversions.
* **Our Story & About Section:** A beautifully formatted brand identity block highlighting the restaurant's background.
* **Interactive Menu Section:** A clean, fully categorized food and beverage display listing descriptions, prices, and dietary tags (e.g., veg/non-veg) designed for seamless scannability.
* **Interactive Photo Gallery:** A visual masonry grid showcasing high-quality restaurant ambiance and featured dishes.
* **Secure Customer Authentication:** Complete login, registration, and Multi-Step Password Reset Flow secured by standard internal PHP Session parameters and interactive OTP (One-Time Password) verification simulations.
* **Table Reservation Pipeline:** A dynamic table booking system where authenticated users can easily request reservations.
* **Direct Contact Gateway:** An input-validated contact form allowing customers to drop direct messages, feedback, or catering inquiries.

### 🛡️ Administrative Control Center
* **Private Admin Dashboard:** A separate, secure login panel restricted exclusively to the restaurant's management team.
* **Real-Time Booking Tracker:** An administrative control panel to view, track, analyze, and manage active customer table reservations.
* **Centralized Inbox:** An internal system that pulls incoming messages and feedback directly from the contact gateway into a clean, actionable grid layout for staff review.

---

## 🛠️ Tech Stack & Architecture

* **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox layouts, fully responsive media queries optimized for Laptop, Tablet, and Mobile screens), JavaScript (ES6+ Form validations).
* **Backend:** PHP (Procedural, object-oriented workflow utilizing native Sessions for state/authentication control).
* **Database:** MySQL (Relational structure managed via phpMyAdmin, handling users, reservation schedules, and message logs).
* **Security Implementations:** SQL Injection protection using database-level escaping, secure data hashing, and granular input lengths.
